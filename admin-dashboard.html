Setup Guide
ขั้นตอนการตั้งค่าระบบ RSVP ทั้งหมด ตั้งแต่ต้นจนถึงส่งการ์ดเชิญ
📋 ขั้นตอนทั้งหมด
1

สร้างฐานข้อมูลบน Railway (Postgres)

สร้าง Postgres service ใน Railway แล้วรัน schema ครั้งแรกจากไฟล์ railway-api/sql/init.sql
จากนั้นเตรียมค่า DATABASE_URL และ LINE_CHANNEL_TOKEN ใน Variables

2

Deploy RSVP API ขึ้น Railway

Deploy โฟลเดอร์ railway-api ขึ้น Railway → ตรวจสอบว่า /health ตอบ { "ok": true }
แล้วคัดลอก URL ของ service มาต่อท้าย /rsvp และใส่ใน Config Tab

3

Upload RSVP Form เป็น LIFF App

ไปที่ LINE Developers Console → เลือก Channel → LIFF → Add LIFF App
สร้าง 2 LIFF แยกกัน:
1️⃣ RSVP LIFF — Size: Full · Endpoint: https://page.sparkth.io/rsvp-form.html
2️⃣ Admin LIFF — Size: Full · Endpoint: https://page.sparkth.io/admin-dashboard.html
เอา LIFF ID ทั้ง 2 ใส่ใน Config Tab

4

ตั้งค่า API URL ในหน้าแอดมิน

เปิดหน้า admin-dashboard.html → แท็บ Config → ใส่ Railway API URL เป็น https://your-app.up.railway.app/rsvp
กดบันทึกแล้ว reload หน้า เพื่อให้ทั้ง admin และ rsvp-form ใช้ endpoint เดียวกัน

5

อัพเดต LIFF URL ใน Flex Message JSON

กลับมาที่ Tab "Flex Message" → คัดลอก JSON → แทนที่ YOUR_LIFF_ID ด้วย LIFF ID ของคุณ → ทดสอบใน Flex Simulator ก่อน

6

ส่ง Flex Message ผ่าน LINE OA

ใน LINE Official Account Manager → Message → Create Message → Custom JSON → วาง Flex Message JSON → ส่งให้แขกได้เลย!
หรือใช้ Push Message API เพื่อส่งแบบ programmatic
