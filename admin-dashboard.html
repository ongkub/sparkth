Setup Guide
ขั้นตอนการตั้งค่าระบบ RSVP ทั้งหมด ตั้งแต่ต้นจนถึงส่งการ์ดเชิญ
📋 ขั้นตอนทั้งหมด
1

สร้าง Google Sheet สำหรับเก็บข้อมูล RSVP

ไปที่ sheets.google.com → สร้าง Sheet ใหม่ → ตั้งชื่อว่า "Wedding RSVP 2026"
ใน Row 1 พิมพ์ header: Timestamp | ชื่อ-นามสกุล | เบอร์โทร | ฝั่ง | ความสัมพันธ์ | จำนวนคน | อาหาร | ข้อความ

2

Deploy Google Apps Script เป็น Web App

ใน Google Sheet → Extensions → Apps Script → วาง code จาก Tab "Apps Script" → Deploy → New Deployment → Web App → Anyone can access → Copy URL → ใส่ใน Config Tab

3

Upload RSVP Form เป็น LIFF App

ไปที่ LINE Developers Console → เลือก Channel (หรือสร้างใหม่) → LIFF → Add LIFF App
Size: Full · Endpoint URL: URL ที่ host rsvp-form.html ไว้
แนะนำ host บน GitHub Pages, Netlify, หรือ Vercel (ฟรี)

4

ใส่ค่าใน rsvp-form.html

เปิดไฟล์ rsvp-form.html → หาส่วน CONFIG → แทนที่:
SHEET_URL ด้วย Apps Script URL
LIFF_ID ด้วย LIFF ID จาก Console → uncomment โค้ด liff.init()

5

อัพเดต LIFF URL ใน Flex Message JSON

กลับมาที่ Tab "Flex Message" → คัดลอก JSON → แทนที่ YOUR_LIFF_ID ด้วย LIFF ID ของคุณ → ทดสอบใน Flex Simulator ก่อน

6

ส่ง Flex Message ผ่าน LINE OA

ใน LINE Official Account Manager → Message → Create Message → Custom JSON → วาง Flex Message JSON → ส่งให้แขกได้เลย!
หรือใช้ Push Message API เพื่อส่งแบบ programmatic

Google Apps Script
วาง code นี้ใน Apps Script ของ Google Sheet แล้ว Deploy เป็น Web App
📋 Apps Script Code (doPost)

1. เปิด Google Sheet → Extensions → Apps Script
2. ลบ code เดิมทิ้ง → วาง code ด้านล่างนี้
3. Deploy → New Deployment → Web App → Execute as: Me · Who has access: Anyone
4. คัดลอก URL ที่ได้ไปใส่ใน Config Tab และในไฟล์ rsvp-form.html

📋 Column Headers สำหรับ Row 1 ใน Google Sheet
Timestamp ชื่อ-นามสกุล เบอร์โทร ฝั่ง ความสัมพันธ์ จำนวนคน อาหาร ข้อความ

คัดลอกแล้ววางใน Row 1 ของ Google Sheet (Tab แรกที่ชื่อว่า Sheet1)
